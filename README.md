# Weather app - Gdańsk University of Technology

A mobile application built with React Native and Expo.

## Requirements

Before running the project, make sure you have:

- Node.js installed, preferably the LTS version,
- npm or yarn installed.

## How to run the project locally?

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd myapp
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
   or:
   ```bash
   yarn install
   ```

### Start the app

Run the local Expo development server:

```bash
npm start
```

or:

```bash
npx expo start
```

This is the standard way to start an Expo project locally.

### Run on iOS

On a Mac, you can run the app on the iOS simulator:

```bash
npm run ios
```

This is equivalent to:

```bash
expo start --ios
```

### Run on Android

Run the app on an Android emulator or a connected device:

```bash
npm run android
```

This is equivalent to:

```bash
expo start --android
```
